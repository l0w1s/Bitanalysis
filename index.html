Sobre o Sir BAP AI

O Sir BAP AI é uma ferramenta de análise técnica automatizada, desenhada para fornecer uma visão clara do mercado de Bitcoin (BTC) com base em estratégias de trading consagradas.

Estratégia Implementada

A aplicação agora utiliza a estratégia Didi Index Improved with QQE.

  • Didi Index: Usa 3 médias móveis (curta, média e longa) para gerar sinais de cruzamento. A força do sinal é confirmada quando os cruzamentos ocorrem acima (para compras) ou abaixo (para vendas) da média central.
  • QQE (Qualitative Quantitative Estimation): Um filtro baseado em RSI aplicado à linha Didi para adicionar uma camada extra de confirmação aos sinais, ajudando a evitar entradas falsas.

Todos os dados de mercado são fornecidos pela API da Hyperliquid. As notícias são fornecidas pela API da CryptoCompare.
